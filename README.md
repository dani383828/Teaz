# Telegram VPN Bot

بات فروش اشتراک VPN با قابلیت‌های کامل مدیریتی

## قابلیت‌ها

### برای کاربران عادی:
- خرید اشتراک VPN با سه پلن مختلف
- افزایش موجودی و مشاهده موجودی
- دریافت کانفیگ VPN
- دعوت دوستان و دریافت پورسانت
- راهنمای اتصال برای دستگاه‌های مختلف

### برای ادمین:
- مدیریت کامل کاربران
- تایید یا رد پرداخت‌ها
- ارسال کانفیگ به کاربران
- آمار و گزارش‌گیری
- مدیریت نمایندگان

### برای نمایندگان:
- خرید اشتراک با قیمت ویژه
- پنل نمایندگی

## نصب و راه‌اندازی

### پیش‌نیازها:
- Node.js 16 یا بالاتر
- PostgreSQL 12 یا بالاتر
- توکن ربات تلگرام
